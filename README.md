# 🏓 Ping Pong Game with Python Turtle

## 📖 Overview
This project was created for fun using **Python’s Turtle graphics library**.  
It recreates a **ping pong game** where two players control paddles to bounce a ball back and forth.  
The game includes **collision detection, scoring, and sound effects** using the `winsound` library.

---

## ✨ Features
- 🎮 **Two-Player Mode**: Player A uses `W` and `S` keys, Player B uses arrow keys.
- 🏓 **Ball Physics**: Ball moves with velocity and bounces off walls and paddles.
- 📊 **Scoring System**: Tracks points for Player A and Player B.
- 🔊 **Sound Effects**: Uses `winsound` for collision feedback (Windows only).
- 🐢 **Turtle Graphics**: Demonstrates real-time updates and interactive gameplay.
